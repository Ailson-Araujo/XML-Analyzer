Base de Dados

Para iniciar o salvamento dos dados na base de dados, antes de tudo, deve-se criar 3 tabelas distintas.

→ TABELA OPERADOR: Graduação, Nome e QM.

→ TABELA CATEGORIA: Categoria em que determinado contado possa se enquadrar.

→ TABELA PONTOS: Categoria da pontuação e valor da pontuação.

Na raiz da aplicação exite a pasta "Json_files" que contém arquivos de configuração que é a base para algumas funções. Tais arquivos podem ser editados caso haja nessecidade.

// Possibilita a escolha do tipo.


[
    "GATO" ,
    "NECI",
    "NERA",
    "FATX",
    "FATM",
    "NIDO",
    "OUTI"
]

// Possibilita a escolha da emissão.


[
    "AM" ,
    "GMDSS",
    "CW",
    "LSB",
    "USB"
]

// Base de dados que determina o quadrante.



{
    "B1":[[-62.5, 10],[-60, 10], [-60, -7.5],[-62.5, 7.5]],
    "B2":[[-60, 10],[-57.5, 10], [-57.5, -7.5],[-60, 7.5]],
    "B3":[[-57.5, 10],[-55, 10], [-55, -7.5],[-57.5, 7.5]],
    "B4":[[-55, 10],[-52.5, 10], [-52.5, -7.5],[-55, 7.5]]

}

Após os dados serem exportados, basta ir na aba de relatório para consultar.
